Create `src/slack/formatters/task-formatter.ts` implementing the three exported functions. Follow the research patterns exactly.

**STATUS_EMOJI map:** `{ open: '⚪', in_progress: '🔵', done: '✅', closed: '⛔', blocked: '🔴', backlogged: '🟡' }`

**PRIORITY_INDICATOR map:** `{ urgent: '🔴 urgent', high: '🟠 high', medium: '🟡 medium', low: '⚪ low' }`

**`formatTaskList(tasks: Array<Task & { tags: string[] }>): KnownBlock[]`**
- Empty → `[{ type: 'section', text: { type: 'mrkdwn', text: '_No tasks found._' } }]`
- Non-empty → HeaderBlock "Tasks (N)" + up to 20 SectionBlocks, each: `${emoji} *#${id} ${title}*\n${priority} · ${assignee || '_unassigned_'}`
- If tasks.length > 20, add a ContextBlock: `{ type: 'context', elements: [{ type: 'mrkdwn', text: 'Showing 20 of ${tasks.length} tasks' }] }`

**`formatTaskDetail(task: Task & { tags: string[] }): KnownBlock[]`**
- HeaderBlock: `{ type: 'header', text: { type: 'plain_text', text: title.length > 147 ? title.slice(0, 147) + '...' : title, emoji: true } }`
- SectionBlock with fields (MrkdwnElement[]): Status, Priority, Assignee (or "_unassigned_"), Due Date (or "_none_"), Project (#id), Created By
- If tags.length > 0, add Tags field: `tags.join(', ')`
- If description not null: DividerBlock + SectionBlock with mrkdwn text (truncated to 3000 chars)

**`formatTaskNotification(event: TaskEvent): KnownBlock[]`**
- EVENT_LABELS map: `{ 'task.created': 'Task created', 'task.updated': 'Task updated', 'task.status_changed': 'Status changed', 'task.claimed': 'Task claimed', 'task.deleted': 'Task deleted' }`
- Single SectionBlock with mrkdwn text joining: event label + actor, emoji + task id/title, priority + assignee, `/tasks show <id>` command

**Type imports:** Import `KnownBlock, SectionBlock, HeaderBlock, DividerBlock, ContextBlock` from `@slack/types` as type-only imports. Import `Task` from `../../types/task.js`. Import `TaskEvent` from `../../events/types.js`.

**Export:** STATUS_EMOJI and PRIORITY_INDICATOR as named exports (project-formatter.ts will reuse them in Plan 02).
