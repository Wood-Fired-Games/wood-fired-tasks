**describe("convertToMcpError"):**

   - Create `new ValidationError({ title: ["Title is required"], priority: ["Invalid value"] })`
   - Call `convertToMcpError(error)`
   - Assert result is instance of McpError
   - Assert result.code === ErrorCode.InvalidParams
   - Assert result.message contains "Validation failed" (or similar)
   - Assert result.data has fieldErrors matching the input

2. "converts NotFoundError to McpError with InvalidRequest code"
   - Create `new NotFoundError("Task", 42)`
   - Call `convertToMcpError(error)`
   - Assert result.code === ErrorCode.InvalidRequest
   - Assert result.message contains "Task" and "42"
   - Assert result.data has entity "Task" and id 42

3. "converts BusinessError to McpError with InvalidRequest code"
   - Create `new BusinessError("Invalid status transition from 'open' to 'done'")`
   - Call `convertToMcpError(error)`
   - Assert result.code === ErrorCode.InvalidRequest
   - Assert result.message contains the business error message

4. "converts unknown Error to McpError with InternalError code"
   - Create `new Error("something unexpected")`
   - Mock console.error to suppress output
   - Call `convertToMcpError(error)`
   - Assert result.code === ErrorCode.InternalError
   - Assert result.message is generic (does NOT contain "something unexpected" -- no info leak)
   - Restore console.error

5. "converts non-Error unknown value to McpError with InternalError code"
   - Call `convertToMcpError("string error")`
   - Assert result.code === ErrorCode.InternalError
   - Assert result.message is generic
