- description: Third-person with triggers. Example: "Creates a new task with configurable project, priority, and assignee. Use when user wants to add a task, create work items, or plan new work."

   - estimated_minutes: Optional.
   - due_date: Optional. ISO 8601 format.
   - tags: Optional. Comma-separated list.
3. Call `wood-fired-bugs:create_task` with all gathered parameters. Always include created_by: 'user'.
4. Confirm creation with task ID, title, priority, and project.

Keep under 80 lines total. Use imperative form.
