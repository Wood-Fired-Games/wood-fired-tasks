## E2E Flow Status

### Flow 1: Full Setup (Linux/macOS)
**Status:** COMPLETE ✓

**Chain:** 
1. User clones repo
2. User runs install.sh
3. install.sh copies skills/tasks/*.md → ~/.claude/commands/tasks/
4. install.sh configures MCP server in ~/.claude.json
5. install.sh validates connectivity to /health endpoint
6. User runs /tasks:log-bug in Claude Code
7. Claude Code loads skill from ~/.claude/commands/tasks/log-bug.md
8. Skill references wood-fired-bugs:create_task and wood-fired-bugs:list_projects
9. Claude Code sends JSON-RPC request to MCP server (node dist/mcp/index.js)
10. MCP server routes to registered tools (task-tools.ts, project-tools.ts)
11. Tools query database at ./data/tasks.db
12. Response returned via stdio (JSON-RPC on stdout)
13. Claude Code displays result to user

**Verified:** All 13 steps traced through code ✓
